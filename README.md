# AI Usage Statement

Este trabalho utilizou uma ferramenta de IA generativa para auxiliar na geração de explicações e
exemplos de código.
Todos os prompts utilizados estão documentados em prompts.txt.
As respostas foram revisadas, testadas e verificadas pelo autor antes de serem incluídas no
relatório.

# Reflexão sobre o uso de IA Generativa

A qualidade do Claude Opus 4.8 me surpreendeu positivamente. Apesar de prompts mais completos 
gerarem, naturalmente, respostas mais completas, mesmo a partir dos prompts mais simples, a LLM
foi capaz de gerar resultados satisfatórios.
A ferramenta é útil para gerar códigos simples e explicar conceitos triviais, além de que se
saiu muito bem na hora de converter sua resposta para LaTeX. No entanto, para casos mais complexos,
é preciso prosseguir com cuidado na hora de utilizar a LLM, por conta que as chances de que ela
alucine são altas. Nesse sentindo, é sempre necessária a verificação das respostas.
Tendo em vista que pode ser uma ferramente muito útil, utilizarei (com cautela) LLMs para maximizar
minha performance em projetos e disciplinas futuras.
